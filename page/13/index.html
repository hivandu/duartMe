32位windows 7 安装64位系统

2010-04-06
develop
32bit, 64bit, System, windows 7

前言:


  • 我并不是为了双系统

  • 因为大部分32位程序在64位Windows 7下运行完美

  • 我是全部从新安装,并无保留任何注册表内容,是否相互兼容未测试!


32位 windows 7中安装64位windows 7会提示与此系统不兼容.

不过利用cmd可以正常安装.

x:\setup /installfrom:y:\sources\install.wim

x,y为代表符,自行更改盘符.”/installfrom”参数指向64位的install.wim，这里的“f:\setup”代表32位Windows 7，而“y:\sources\install.wim”则表示来自64位Windows 7的安装镜像（具体盘符根据个人电脑不同可能有差别，大家自行修改吧）.有不懂的留言!

PS:无法升级,两个系统相互之间不兼容!只能从新安装.

PS2:AE对一个15秒的动画序列帧已经渲染了两个多小时了,极度无语.明天十点之前还要出门,打算不睡觉了!

用comodo 限制TM

2010-03-25
software
TM, comodo, 安全, 限制

最近都在研究HIPS,说实话,这是一个比较麻烦的事情!

而我用Comodo的主要目的就在于自己的NOD32到期后可以单奔,但是暂时,我仅仅想对腾讯对我的侵犯进行限制!

说明一下,为了安全和避免麻烦,最好选用TM.如果选用QQ,对应规则还是一样的,只是会不断的弹出selfupdate.exe对话框,不嫌麻烦的就尽量去点吧!以下规则在TM中测试完全没问题,正常使用,但是在QQ中就无法访问自己的文件夹,就算添加了允许文件夹列表也不行,必须完全允许QQ访问文件夹,而在Comodo日志中也找不到相应的访问记录!所以,用QQ的可以考虑将受保护文件/目录设为允许,可是安全性就大打折扣!
